# NOMENCLATURA CSS
 
## BEM
 
1. Block (Bloque): Componente y/o módulo independiente.
2. Element (Elemento): Cada elemento que conforma el bloque.
3. Modifier (Modificador): Una variable del bloque que cambia una propiedad del elemento.

```
.bloque {...}
.bloque__elemento {...}
.bloque--modificador {...}¨
```

.menu {...}
.menu__item {...}
.menu__item--active {...}



## SUITCASE
